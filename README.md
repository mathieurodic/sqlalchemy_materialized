# etl-decorators

A collection of small, focused decorators used in ETL / data applications.

## Installation

Base package (no optional dependencies):

```bash
pip install etl-decorators
```

SQLAlchemy subpackage:

```bash
pip install etl-decorators[sqlalchemy]
```
